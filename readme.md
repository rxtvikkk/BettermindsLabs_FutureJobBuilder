# Future Job Builder 🚀

Future Job Builder is an AI-powered career discovery platform tailored for high school students. Unlike standard career quizzes, it evaluates interests, personal strengths, work style preferences, and lifestyle priorities dynamically to output a robust career profile. 

Crucially, it implements an educational **AI Resistance Score** to guide students on skills that are highly automation-resistant, alongside actionable milestone roadmaps.
